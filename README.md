# React-native-dynamic-module-poc
 Dynamic delivery is a android feature , and is not directly supported by React-native.
 This Poc aims at :
 1) creating a dynamic feature module
 2) download and access its resources in this RN app

## Installation
```sh
$ yarn install
```
For Android
```sh
$ yarn android
```
For ios
```sh
$ cd ios && pod install
$ cd .. && yarn ios
```
